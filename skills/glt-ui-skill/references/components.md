# glt-ui component inventory

All names export from `import { … } from 'glt-ui'`.

## Atoms

| Component | Role |
| --- | --- |
| Avatar | User/entity image or initials |
| Badge | Status / count pill |
| Block | Typography block spacing |
| Box | Generic padded box |
| Button | Primary actions (`variant`, `size`, `loading`) |
| Checkbox | Boolean input |
| Code | Inline code |
| Container | Page width constraint |
| Content | Prose content wrapper |
| CountUp | Animated number |
| DeleteButton | Destructive icon/button |
| Divider | Horizontal rule |
| Grid / GridItem | CSS grid layout |
| Heading | Semantic headings |
| Icon | Glyph wrapper |
| ImagePlaceholder | Placeholder media frame |
| Label | Form label |
| Level / LevelLeft / LevelRight / LevelItem | Horizontal level layout |
| List / ListItem | Lists |
| Markdown | Markdown renderer |
| MediaObject | Media + content row |
| ProgressBar | Linear progress |
| ProgressRing | Circular progress |
| Radio | Radio input |
| Reveal | Scroll-in motion |
| SelectField | Select control |
| SiteNavLink | Header nav link |
| Skeleton | Loading placeholder |
| Sparkline | Tiny trend chart |
| Spinner | Loading spinner |
| Stack | Flex stack with gap |
| Surface | Elevated surface |
| Switch | Toggle |
| Text | Body text (`size`, `muted`, etc.) |
| TextArea | Multiline input |
| TextInput | Text input |
| Title / Subtitle | Display titles |
| Tooltip | Hover hint |
